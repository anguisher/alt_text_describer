Generates alternative text for images using AI. Fully automatic process without any effort.

== Installation ==
Simply install Wordpress plugin.
Register at https://prisakaru.lt and buy one of our credits package.
In your account on https://prisakaru.lt you will find API Key which you will need to put it on plugin \"Settings\" tab.
